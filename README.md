Static project page for:

**PhaForce: Phase-Scheduled Visual-Force Policy Learning with Slow Planning and Fast Correction for Contact-Rich Manipulation**

Accepted to IROS 2026.

## Public Deployment

This site is ready for GitHub Pages.

1. Create a public repository, for example `phaforce`, under `https://github.com/Thu-WangMX`.
2. Push this folder to the repository.
3. In GitHub, open `Settings -> Pages`.
4. Set `Build and deployment` to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.

Example push commands:

```bash
git branch -M main
git add .
git commit -m "Add PhaForce project page"
git remote add origin git@github.com:Thu-WangMX/phaforce.git
git push -u origin main
```

The public URL will be:

```text
https://thu-wangmx.github.io/phaforce/
```

If you use the special repository name `Thu-WangMX.github.io`, the URL will be:

```text
https://thu-wangmx.github.io/
```
